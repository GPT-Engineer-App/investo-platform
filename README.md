# investo-platform

I want to build a web application similar to a web-based investment brokerage platform, inspired by features on schwab.com. The platform should cater to individual investors and offer a variety of investment options and functionalities.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/investo-platform.git
cd investo-platform
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
